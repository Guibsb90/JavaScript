Eventos no JS

Para trabalhar com eventos no JS, temos que referenciar os elementos no HTML com ID's ou Classes

Após criar uma variável para guardar o id ou class, chame a variável e adicione .addEventListener() e adicione os parametros no caso desse botão na pagina, um parametro 'click' que chama a função de clickar, por exemplo e uma função para poder ditar o comportamento do botão ou podemos fazer a função separadamente e somente chama-la como parametro dentro dos parenteses do addEventListener.
