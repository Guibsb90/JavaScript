ES Modules

Para usar o ES Module basta acrecentar a palavra export antes da palavra function para exportar o objeto. Ex:

   export function exemplo(exemplo){
        return a+b
        }

Para importar o arquivo, usamos a palavra import com chaves {} e colocar o nome dos objetos que foram exportados e apos fechar a chave, usamos a palavra from e o local de onde est√° o arquivo entre aspas.E, por fim, temos que ir no HTML e passar na tag script um atributo TYPE=module. Ex: 

        import {name, label} from "./functions.js"

  E, no html:

      script type="module">exemplo.js
