JavaScript

Cliente é o computador, o celular. O servidor fornece os dados para podemos acessar e os clientes são quem querem os arquivos. Js é uma tecnologia client side.

Com Js podemos fazer varias mudanças com poucos comandos. A tag HTML para se adicionar o JS é a tag SCRIPT. Dentro dela que será desenvolvido o código em JS.

Um sinal de = no JS significa RECEBE. ex: vaga a1= carro1 (vaga a1 recebe carro a1). A palavra NULL é um atributo que indica que não tem nada na variável (VAR). Quando um valor é colocado dentro de uma variável chamamos de ATRIBUIÇÃO.

As variáveis chamam-se IDENTIFICADORES. Pode começar com letras, $ ou _. Não podem começar com números. É possivel usar letras ou números. É possível usar acentos e símbolos. Não pode conter espaços. E, não pode ser palavras reservadas. Para botar palavras como valor das variaveis, é necessário usar ASPAS ', " ou `crase. Uma STRING é uma informação em letras, cadeias de caracteres e combinação de numeros, como um CPF, por exemplo. Números inteiros, números reais são chamados de number Valores como true e false são chamados de boolean. Esses são os 3 tipos primitivos principais. Mas, existem muitos outros tipos primitivos.

Para abrir o NodeJs, basta abrir o Visual Code e clicar em TERMINAL, na parte de cima.

Tipos Primitivos

  • Number - Infinity, NaN
  • String (uma informação entre ", ', `)
  • Boolean
  • Null
  • Undefined
  • Object - Array (uma informação entre [], {},
  • Funcion

Transformando string em número

Para converter uma string em números podemos usar Number.parseInt(numero), que converte numeros inteiros. E, Number.parseFloat(numero) para converter numeros reais (com pontos) ou ainda pode-se usar somente Number. .Olhe codigo fonte pra maiores explicações.

Transformando número em string

Para converter um numero em string podemos usar String(valor) ou (o que eu quero converter)toString().
