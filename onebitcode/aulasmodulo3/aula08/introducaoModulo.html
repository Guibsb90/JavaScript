Introdução aos Módulos

Um módulo é uma forma de agrupar arquivos de JS para que não fique todo código agrupado num unico arquivo. Facilita para mautenção do código e para identificação dos problemas e das funcionalidades que cada um tem. Também tem como compartilhar os módulos.

Os módulos podem ser reaproveitados em outros projetos diferentes que necessitem de configurações iguais as que o módulo possui.
