Operador Spread

O operador spread permite espalhar os elementos de um objeto iterável, como um array ou uma string.

Para utilizar o operador spread basta botar 3 pontos depois do nome do objeto. 

            Ex:
            let srt = "ola mundo"
            let arr = [1,2,3]

            console.log(...str) (vai retornar assim: o l a , m u n d o )
            console.log(...arr) (vai retornar assim: 1 2 3)
            E para clonar um objeto, pode fazer dessa forma:

            let clone = [...srt]
            Desta forma, terá feito uma copia
