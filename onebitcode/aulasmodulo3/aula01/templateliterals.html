Template Literals

O Template Literals é um outro modo de se declarar strings.

Para usa-lo, use crases no lugar das aspas (`). Usando template, se pular a linha, ele pula também e para adicionar strings ou contas, basta usar ${} e escrever dentro.

Ex: 

                let string = `Desta forma,
                essa quebra de linha e ${nome} a variavel nome
                serão automáticas.`
