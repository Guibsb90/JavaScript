Condições

Digite a Velocidade:

If (se) e Else (se não)

Para criar um bloco de condição, escreva if(condição){ } e para criar o outro else{}.

Tipos de Condição

Condição simples - if (condição) { true }. Só tem um bloco e caso a condição seja falsa, não acontecerá nada.

Condição composta - if (condição) { true } num bloco e else (condição){} no outro
