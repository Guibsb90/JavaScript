Array high order function

Um array high-order function é uma função que trata um array (ou lista) como entrada e retorna um novo array como saída, aplicando uma transformação ou operação a cada elemento do array original. Em outras palavras, essas funções tratam o array como uma coleção de elementos que podem ser manipulados e transformados em outros elementos.

Exemplos de funções de ordem superior em arrays incluem o método map, que aplica uma função a cada elemento do array e retorna um novo array com os resultados, o método filter, que retorna um novo array contendo apenas os elementos que satisfazem uma determinada condição, e o método reduce, que combina todos os elementos do array em um único valor, usando uma função acumuladora.
