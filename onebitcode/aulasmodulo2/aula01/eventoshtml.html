Eventos no HTML

Os eventos são uma forma dinâmica de executar js dentro de uma página HTML.


Permite executar o js sem travar o carregamento da página, como num alert ou um prompt.


São acionados por ações específicas na página (clicar, passar o mouse, focar, digitar, etc)


Um dos eventos é o onmouseenter, que inicia uma função quando o cursor do mouse vai para algum lugar especifico


O evento onchange , que funciona após preencher um input, por exemplo, e depois clicar do lado de fora.


O evento onfocus é acionado quando o elemento é focado, seja com o mouse ou com tab.


O elemento onblur é acionado quando o cursor sai do elemento


O elemento onclick é para clicar
