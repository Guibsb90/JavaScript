Criando elementos DOM

O DOM nos permite adicionar dinamicamente novos elementos HTML ao docmento. Para isso, precisamos criar o elemento, que podemos fazer utilizando um document.createElement().

Para adicionar conteúdo ao elemento, usamos newElement.innerHTML =

Para adicionar o elemento como um novo nó no DOM: element.appendChild(newElement)

Também é possivel clonar um elemento já existente.
