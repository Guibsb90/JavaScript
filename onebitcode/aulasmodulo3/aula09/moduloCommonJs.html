Módulos CommonJS

Existem duas formas de se trabalhar com módulos em Js. Uma delas é com CommonJS.

Para utilizar os módulos com CommonJS, se criam 2 ou mais arquivos para cada função
ou blocos de funções existentes em cada módulo e é necessário exportar o módulo, dessa forma:

      module.exports = (nomeDoModulo)

Para chamar o módulo que exportamos, precisamos colocar o módulo numa variavel, dessa forma:

      const (nomeDaVariavel) = require(""./render") onde a função require recebe como 
      parâmetro o local onde está armazenado o módulo que iremos usar.

Exemplo:

      
        const render = require("./render")
        const store = require("./require")
        console.log("Aplicação iniciada")
        render()
        store()
        console.log("Aplicação finalizada.")
      
Desta forma, os modulos render e store serão chamados de seus respectivos arquivos para funcionar em outro arquivo.
