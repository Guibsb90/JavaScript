Arrow Function

As funções anonimas não tem nome, portanto, são criadas dentro de uma variável. Arrow functions são uma forma diferente de escrever funções anonimas e não utiliza a palavra function no inicio e sim os simbolos => no inicio, por isso o nome arrow.

A arrow function não possue as propriedades this, arguments, super e new.target. Em alumas situações isso pode ser relevante.

A arrow function permite retornar diretamente o valor de uma função se for apenas uma linha sem necessitar do return. ex: 

                let multiply = (a,b) => a*b
