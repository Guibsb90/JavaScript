Do While

A estrutura de repetição do while é muito parecido com o while, porém ele garante que o bloco de código será executado pelo menos umam vez, mesmo que a condição seja falsa.

No do while a primeira verificação da condição só acontece depois que o bloico foi executado uma vez.

FAÇA alguma coisa ENQUANTO condição.

Sintaxe:

            var resultado = '';
            var i = 0;
            do {
                i += 1;
                resultado += i + ' ';
            } while (i < 6);
            document.innerHTML = resultado;
