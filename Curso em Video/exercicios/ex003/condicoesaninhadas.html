Condições Aninhadas

Como fazer

Para fazer uma condição aninhada, você vai utilizar uma condição dentro da outra do jeito que está descrito abaixo: 

                if (cond1){
                    bloco 1
                } else {
                    if(cond2){
                        bloco2
                    }else{
                        bloco3
                    }
                }

Condições Múltiplas

A condição múltipla consegue expressar mais valores que só sim ou não. Exemplo abaixo:

                switch (expressão){
                    case valor1:
                        (     )
                        break
                    case valor2:
                        (     )
                        break
                    case valor3:
                        (     )
                        break
                    default:
                        (     )
                        
                }
