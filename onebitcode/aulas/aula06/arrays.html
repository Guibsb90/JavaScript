Arrays

É uma estrutura do JS para se trabalhar com too tipo de lista.

Seus elementos são identificados por um índice inteiro começando pelo 0.

pode armazenar qualquer tipo de dado e ter qualquer tamanho

pode ser criado e utilizado atravéz de colchetes []

Possuem funções para vários casos de uso.

Para adicionar um elemento no array (no final) com a função .push("nome do elemento"). EX: nomedoarray.push("elemento")

A função .unshift() serve para colocar elementos no inicio do array.

Para remover elementos de um array, podemos usar a função .pop() (remove o ultimo elemento do array)

Para remover elementos do inicio do array, usamos a função .shift()

Para verificar se um elemento está no array, usamos .includes()

Para saber o indice de um elemento no array, usamos .indexOf()

Para dividir e pegar uma parte do array e salvar em outro, usamos .slice(0,4){os valores são referencias de quais indices serão usados para fazer a copia dos elementos. O último numero tem que ser subtraido de 1. No exemplo dado, iria da 1 a 4 posiçao , lembrando que os arrays começam na posiçao 0 }

Para concatenar arrays, usamos .concat()

Para remover um grupo de elementos do array, usamos .splice(indice, 1,, "gendalf, o Cinzento")Dentro, botamos o numero do indice e a quantidade que é pra ser excluida a partir dele, e, depois, o que irá substitui-lo.
