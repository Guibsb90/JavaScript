Desestruturação de Array e Objeto

A desestruturação nada mais é que extrair valores de forma mais conveniente de objetos e arrays, atribuindo-os a variáveis individuais.

Para fazer a desestruturação, basta fazer como se estivesse criando uma variavel, porém, deve-se botar chaves para o objeto e colchetes para o array, dessa forma: 

            let [a,b] = number

No caso, o 'number' é o nome do array que está sendo desestruturado.

Para desestruturar o array, ele vai pegar os elementos na ordem que estão dispostos no array.
