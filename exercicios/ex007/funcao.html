Funções são ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função pode receber parâmetros e retornar um resultado.

Dentro dos () parenteses do nome da fução nós adicionamos parâmetros.

        function ação(parâmetro){
                                 < -  (ação fica dentro das chaves{})
            return res(retorno)
        }
        ação(5) (chamada)
